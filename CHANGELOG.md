# Changelog

Todas as mudanças notáveis deste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-01-15

### Adicionado
- Sistema completo de gerenciamento de armários (lockers)
- Painel de controle com métricas e gráficos em tempo real
- Autenticação com JWT e controle de sessão
- Gerenciamento multi-empresa (multi-tenant)
- Cadastro de funcionários/clientes com importação em massa
- Sistema de reservas com contagem regressiva e expiração automática
- Fila de espera inteligente para portas ocupadas
- Sistema de renovação de reservas com aprovação administrativa
- Notificações por e-mail e WhatsApp (integração Uazapi)
- Templates personalizáveis de notificações
- Personalização visual (white-label) por empresa
- Portal do usuário para autoatendimento
- Auditoria completa de ações do sistema
- Departamentos e setores organizacionais
- Histórico detalhado de uso de portas
- Sistema de auto-atualização via Git/Docker
- Rollback automático em caso de falha na atualização
- Suporte a temas claro/escuro
- Layout responsivo para dispositivos móveis
